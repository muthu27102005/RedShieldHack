# AI Crime Prediction Web App

A functional Python-based web app that allows CSV uploads, processing, dynamic Folium maps, and KMeans clustering for crime hotspots.

## Folder Structure
- `app.py`: Main backend entry point
- `data/`: Contains datasets
- `utils/`: Core processing logic (Data, Model, Map)
- `templates/`: HTML files
- `static/`: CSS and JS resources
- `models/`: Saved ML models

## To Run
1. `pip install -r requirements.txt`
2. `python app.py`
